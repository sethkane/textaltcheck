Text Alternative Checker

This feature toggles the Text Alternatives Checker to the on/off position for images found on the page. All background images will be hidden when enabled.

Key:

An image with a link wrapped around it. Reference: functional image
An image with a link wrapped around it that has a null value. Reference: functional image
An image with a link wrapped around it that has a text alternative which needs review for context, reducancy, and other considerations. Reference: functional image, fink purpose
